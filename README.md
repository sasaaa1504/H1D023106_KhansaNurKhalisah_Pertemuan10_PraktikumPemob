## Penjelasan Fungsi
Aplikasi manajemen produk berbasis Flutter dengan fitur CRUD (Create, Read, Update, Delete):
- **LoginPage**: Form login dengan validasi email dan password, terdapat link navigasi ke halaman registrasi.
- **RegistrasiPage**: Form pendaftaran dengan validasi nama (min 3 karakter), email (format valid), dan konfirmasi password (min 6 karakter).
- **ProdukPage**: Menampilkan daftar produk dalam bentuk Card dengan informasi kode, nama, dan harga yang dapat diklik untuk melihat detail.
- **ProdukDetail**: Menampilkan detail produk lengkap dengan tombol Edit dan Delete untuk memodifikasi atau menghapus data.
- **ProdukForm**: Form dual-purpose untuk menambah produk baru atau edit produk existing dengan validasi input lengkap.
